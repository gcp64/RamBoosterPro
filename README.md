**RamBooster Pro** is a professional-grade Windows system optimizer featuring a stunning glassmorphic UI with interactive constellation particle background, real-time memory monitoring, HWiNFO-level BIOS & hardware diagnostics, Roblox & Genshin Impact FPS boost, deep disk storage cleaning, and an advanced Telecom & IT network suite.

**RamBooster Pro** هو محسّن نظام ويندوز احترافي بواجهة زجاجية فاخرة مع خلفية نجوم وتأثيرات تفاعلية مع الماوس، مراقبة ذاكرة حية، فحص دقيق للبايوس والقطع، وضع ألعاب لتعزيز Roblox و Genshin Impact، تنظيف عميق للقرص، وجناح متقدم لأدوات وفحص الشبكات.

## Features | المميزات

### 🖥️ Deep Hardware & BIOS Suite (HWiNFO / CPU-Z Engine) | فحص العتاد والبايوس
- **BIOS & Motherboard**: Vendor, Version, Release Date, UEFI status, Secure Boot state, Motherboard Manufacturer, Model & Serial Number.
- **CPU Microarchitecture**: Model name, Physical Cores, Logical Threads, Max Clock Speed (MHz), L2/L3 Cache size, Virtualization (VT-x / AMD-V) status.
- **RAM Memory Sticks**: Total Capacity, Slots Used vs Available, Speed (MHz), Generation (DDR4/DDR5), per-slot stick details (Slot, Manufacturer, Speed).
- **GPU & Display Engine**: Dedicated VRAM, Driver Version, Screen Resolution & Refresh Rate (Hz).
- **Storage & Bus**: SSD/HDD models, Size (GB), Bus Type (NVMe PCIe / SATA).
- **Windows Kernel**: Edition, System Uptime, TPM 2.0 Security Chip status, Antivirus status.

### 🌐 Telecom & IT Network Suite | جناح وإداريات الشبكة والاتصالات
- **Live Gaming PingPlotter & Hop Tracer**: Traces network route node-by-node (Roblox, Genshin, EU, ME) to identify lag bottlenecks per hop.
- **Wi-Fi Spectrum & Channel Congestion Analyzer**: Scans surrounding Wi-Fi networks, RSSI (dBm), signal percentage, and recommends optimal un-congested channels.
- **LAN Device Inspector & Fingerprinter**: Discovers all active devices connected to the router with hostname, IP, MAC address, and vendor identification.
- **Live Internet Speedtest Engine**: Real-time Download, Upload (Mbps), and Ping latency test.
- **DNS Leak & Encrypted DoH Inspector**: Checks for ISP DNS leaks and activates Encrypted DNS-over-HTTPS (DoH).
- **Real IP Configurator**: Static IPv4 setup & DHCP switching.
- **MAC Spoofer & Proxy Switcher**: Randomizes MAC address & configures system proxy.
- **Subnet Calculator (CIDR/IPv4)**: Calculates Netmask, Network IP, Broadcast IP, and Host limits.
- **DNS Switcher & Benchmark**: Benchmarks Cloudflare, Google, Quad9, AdGuard, OpenDNS.
- **Active Socket Auditor**: Live NETSTAT connection inspector linked to Process IDs.

### 🎮 Game Mode & Roblox / Genshin Ultra Boost | وضع الألعاب الخاص
- **Roblox & Genshin Impact Exclusive Boost**: Configures High-Performance Discrete GPU rules, disables GameDVR stuttering, and boosts process priority.
- **System Game Mode**: Stops non-essential background Windows services and activates High Performance power plan.

### 💾 Storage & Disk Cleaner | أدوات وتطبيق القرص
- **Multi-Drive Partitions**: Multi-drive space scanner (C:, D:, E:).
- **Deep Disk Cleaner**: Clears System Temp, Browser Caches (Chrome, Edge, Firefox, Opera, Brave), and Windows Update leftovers.
- **Storage File Categorizer**: Calculates space used by Videos, Music, Photos, Archives, and Documents.
- **Large File Finder**: Scans for files larger than 30MB for deletion.
- **Installed Applications Uninstaller**: Views and launches uninstallers for installed applications.
